Plane Travel Map

Carbon Emissions for Flights

We got 2008 flight pattern data from: Mapping Airport Connections Tutorial. Using a distance calculator we were able to find the distance between selected airports. The following process allows us to visualize carbon emissions from these patterns: 90. 7 Passengers per flight * count flight * flight distance * 285g of CO2/passenger/km = total CO2 emission, in metric tonnes, of each flight route over a year.

Using this information, we created a map visualization to explore the carbon footprints of these flight patterns. Color shows the carbon footprint from flight route and line thickness is related to the number of flights that occured.

There is less travel to airports furthest away from INDY resulting in a smaller carbon footprint. From this visualization we learned that using an airplane for transportation uses more carbon footprint than we expected.

Barchart

Emissions for Different Methods of Transportation

The data used for this visualization is from the European Environment Agency. In this visualization, we look at eight different methods of transportation: small petrol car, medium petrol car, large petrol car, small diesel car, large diesel car, bus, two-wheeler, and an aircraft. The average number of passengers for each method of transportation varies. The averge number of passengers for cars, busses, and airplanes are: 1.52, 12.7, and 88.

The focus of this visualization is on the carbon emissions for the average number of passengers for each method of transportation.

From this visualization, we can conclude that travel by airplane produces the most amount of CO2 per person per kilometer traveled whereas travel by bus produces the least amount of CO2 per person.

To create this visualization we modified the code, BarChart with ToolTip.
